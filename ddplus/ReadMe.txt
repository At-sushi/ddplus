========================================================================
    スタティック ライブラリ : ddplus プロジェクトの概要
========================================================================

AppWizard はユーザー用に ddplus ライブラリ プロジェクトを作成しました。
このファイルには ddplus アプリケーションを構成している
各ファイルの概要が含まれています。


ddplus.vcproj
    これはアプリケーション ウィザードで生成された VC++ プロジェクトのメイン プロジェクト ファイルです。
    ファイルが生成された Visual C++ のバージョン情報が含まれています。また、 
    アプリケーション ウィザードで選択したプラットフォーム、構成およびプロジェクト機能に関する
    情報も含まれています。


/////////////////////////////////////////////////////////////////////////////

StdAfx.h, StdAfx.cpp
    これらのファイルはプリコンパイル済みヘッダー (PCH) ファイル ddplus.pch、
    およびプリコンパイルされた型の (PCT) ファイル stdafx.obj をビルドするために使われます。

必要な場所を示します。
